Datasets (ImageNet 2012)

The ImageNet validation dataset will be used for this job template.

Download and extract the ImageNet2012 validation dataset from http://www.image-net.org/ into the imagenet/val directory, then move validation images to labeled subfolders, using the valprep.sh shell script. Operating as follows:

cd imagenet/val
wget https://raw.githubusercontent.com/soumith/imagenetloader.torch/master/valprep.sh
bash valprep.sh

A after running the data prep script, your folder structure should look something like this:
