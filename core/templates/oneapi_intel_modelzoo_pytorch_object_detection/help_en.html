Datasets (COCO 2017)

Download the 2017 COCO dataset using the download_dataset.sh script. Operating as follows:

cd <path to your clone of the model zoo>/quickstart/object_detection/pytorch/maskrcnn_resnet50_fpn/inference/cpu
export DATASET_DIR=<directory where the dataset will be saved>
bash download_dataset.sh

Note: The <intel_oneapi_path>/modelzoo/latest/models path holds a Model Zoo project.
