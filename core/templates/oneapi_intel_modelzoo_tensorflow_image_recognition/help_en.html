For TensorFlow Image Recognition of Intel Model Zoo, Datasets need to be prepared in some cases. see the Download and preprocess the ImageNet dataset website to obtain the dataset.

The final Dataset format should look like the following:

$ ls -1 < IMAGENET DIR >/tf_records/

train-00001-of-01024
train-00002-of-01024
train-00003-of-01024
...
