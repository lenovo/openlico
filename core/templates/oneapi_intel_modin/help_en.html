To use Modin to speed up Pandas, you need to modify the Pandas code by simply before running the job and use Modin just like Pandas

Before modification

  1. import pandas as pd
  2. ...

After modification

  1. import modin.pandas as pd
  2. ...

Modin code example

  1. import modin.pandas as pd
  2. import time
  3. s = time.time()
  4. df = pd.read_csv("test.csv")
  5. e = time.time()
  6. print("Modin read time = {}".format(e-s))
