帮助

使用Modin加速Pandas运行，您只需要在运行作业前，简单的修改您的Pandas代码，就可以像使用Pandas一样使用Modin了

修改前

  1. import pandas as pd
  2. ...

修改后

  1. import modin.pandas as pd
  2. ...

Modin 代码示例

  1. import modin.pandas as pd
  2. import time
  3. s = time.time()
  4. df = pd.read_csv("test.csv")
  5. e = time.time()
  6. print("Modin read time = {}".format(e-s))
