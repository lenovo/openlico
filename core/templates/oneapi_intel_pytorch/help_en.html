To use Intel GPU for your pytorch program, If your program contains code like torch.device("cpu"), you should make corresponding simple modifications before running the job.

Before modification

  1. device = torch.device("cpu")
  2. ...

After modification

  1. import intel_extension_for_pytorch as ipex
  2. device = torch.device("xpu")
  3. ...

Intel PyTorch with Intel GPU code example

  1. import intel_extension_for_pytorch as ipex
  2. import torch
  3. import torchvision.models as models
