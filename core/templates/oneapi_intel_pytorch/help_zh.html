帮助

在PyTorch程序上使用Intel GPU，如果您的程序包含类似 torch.device("cpu")的代码，则在运行作业之前应进行相应的简单修改。

修改前

  1. device = torch.device("cpu")
  2. ...

修改后

  1. import intel_extension_for_pytorch as ipex
  2. device = torch.device("xpu")
  3. ...

代码示例

  1. import intel_extension_for_pytorch as ipex
  2. import torch
  3. import torchvision.models as models
  4. device = torch.device("xpu")
  5. model = Net()
  6. model = model.to(device)
